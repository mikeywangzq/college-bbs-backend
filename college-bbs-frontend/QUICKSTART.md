# 快速开始指南

## 🚀 5分钟快速启动

### 1. 前置条件检查

确保已安装：
```bash
node --version  # >= 16.0.0
npm --version   # >= 8.0.0
```

### 2. 后端准备

确保后端服务正在运行：
```bash
# 在另一个终端，启动后端
cd ../college-bbs
./build/college-bbs config.json
```

后端应该运行在 `http://localhost:8080`

### 3. 前端启动

```bash
# 进入前端目录
cd college-bbs-frontend

# 安装依赖（首次运行）
npm install

# 启动开发服务器
npm run dev
```

### 4. 访问应用

打开浏览器访问：http://localhost:5173

## 🛠️ 常见问题

### Q: 启动后页面空白？
A: 检查浏览器控制台，可能是API地址配置错误。确认`.env`文件中的`VITE_API_BASE_URL`正确。

### Q: 登录后刷新页面退出？
A: 正常情况不会退出，Token存储在localStorage中。如果退出，检查浏览器是否禁用了localStorage。

### Q: API请求失败？
A: 
1. 确认后端服务是否正在运行
2. 检查后端是否配置了CORS
3. 查看浏览器Network标签查看具体错误

### Q: 点赞/回复没有反应？
A: 确保已登录，未登录用户无法点赞和回复。

## 🔧 开发提示

### 修改API地址

编辑 `.env` 文件：
```bash
VITE_API_BASE_URL=http://your-api-server:port
```
